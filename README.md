## Sobre

API para carrinho inteligente com IA, desenvolvida com NestJS, Prisma, Google GenAI e testes automatizados.

O objetivo deste repositorio e demonstrar uma implementacao clara, organizada e facil de avaliar, com documentacao objetiva para recrutadores e pessoas tecnicas entenderem rapidamente o contexto do projeto.

## Funcionalidades

- Endpoints para carrinho.
- Integracao com IA generativa.
- Persistencia com Prisma.
- Seed de banco.
- Testes unitarios e E2E.
- Validacoes de dados.

## Stack

- **NestJS**
- **TypeScript**
- **Prisma**
- **Google GenAI**
- **Zod**
- **Jest**

## Como executar

```bash
git clone https://github.com/marquesmaycon/cart-ai-api.git
cd cart-ai-api
npm install
npm run dev
```

## Scripts disponiveis

- `npm run build`
- `npm run format`
- `npm run start`
- `npm run start:dev`
- `npm run start:debug`
- `npm run start:prod`
- `npm run lint`
- `npm run test`
- `npm run test:watch`
- `npm run test:cov`

## Destaques tecnicos

- Backend moderno com NestJS.
- Uso de IA em regras de produto.
- Boa base para demonstrar testes e arquitetura API.
